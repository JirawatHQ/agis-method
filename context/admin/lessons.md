# Lessons Ledger (ยิ่งผิด ยิ่งเก่ง)

บันทึกทุกครั้งที่: ผู้ใช้ correct งาน · ผู้ช่วยจับความพลาด/ช้า/เปลือง token ของตัวเอง · subagent ส่งงานคุณภาพต่ำ
ทุก lesson ต้องปิดด้วย patch อย่างน้อย 1 ชั้น (hook/skill/CLAUDE/memory/wiki) — ไม่มี patch = ยัง `open`

## รูปแบบ
```
### [YYYY-MM-DD] อาการสั้นๆ
- **อาการ:** เกิดอะไร
- **root cause:** ทำไมถึงพลาด
- **patch:** แก้ที่ชั้นไหน (ไฟล์ไหน) — [open | done]
```
